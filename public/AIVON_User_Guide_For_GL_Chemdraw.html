NMR & Spectrum Analysis

Why does the assistant zoom on the labels?
The system creates 5 zoomed crops of the label regions (top strip, downfield, upfield, center, and lower band) to ensure the AI reads the actual printed ppm values. This prevents common OCR errors where the AI might guess values like 160.87 instead of reading the printed 160.07. Each crop is also enhanced with contrast boosting and sharpening for better text visibility.
What is 3-pass verification?
A three-step process to maximize accuracy:
  1. Pass 1 ‚Äî Initial read: The AI reads every printed ppm label visible in each of the 5 crop images
  2. Pass 2 ‚Äî Cross-crop agreement: Values are compared across crops. A ppm value is only confirmed if the identical value appears in 2 or more crops. Single-crop values are flagged as unconfirmed
  3. Pass 3 ‚Äî Digit verification: Each confirmed value is re-examined digit by digit, checking for commonly confused pairs (0‚Üî8, 1‚Üî7, 2‚Üî7, 3‚Üî5, 6‚Üî8) and verifying decimal point placement
How accurate is the peak reading?
The zoomed label reading and 3-pass verification help improve accuracy on printed labels. The AI reports confidence levels for each peak:
  • HIGH ‚Äî Value confirmed in 3+ crops
  • MEDIUM ‚Äî Value confirmed in 2 crops
  • LOW ‚Äî Value found in only 1 crop (flagged as unconfirmed)
Only HIGH and MEDIUM confidence values are included in the ACS-style summary.
Tips for best spectrum analysis results?
  • Use high-resolution images ‚Äî The clearer the printed labels, the more accurate the reading
  • Make sure labels aren't cut off ‚Äî Include the full label region at the top of the spectrum
  • Specify the nucleus ‚Äî Tell the AI what type of spectrum it is (e.g., "This is a 13C NMR" or "Analyze this 1H spectrum")
  • Mention the solvent ‚Äî If visible, mention the solvent (CDCl‚ÇÉ, DMSO-d‚ÇÜ, etc.) so the AI can properly exclude solvent peaks
  • Ask follow-ups ‚Äî After the initial analysis, ask about specific peaks, functional group assignments, or structural interpretation
  • Upload multiple spectra ‚Äî You can upload ¬πH and ¬π¬≥C together and ask the AI to cross-correlate
What output do I get from a spectrum analysis?
The AI provides:
  • A brief identification (nucleus type, solvent, 1D vs 2D)
  • A complete peak table with ppm values, confidence levels, and which crops confirmed each peak
  • An ACS-style summary using proper notation (Œ¥, ŒΩÃÉ, Œªmax) ‚Äî ready for publication
  • Structural insights and tentative functional group assignments
You can then export to pdf or export to word to save the analysis as a formatted report.

Troubleshooting

File upload not working?
Check that your file is under 50MB and in a supported format. Try refreshing the page if issues persist. For very large files, try splitting them. PowerPoint files up to 50 slides are supported.
Spectrum analysis seems wrong?
  • Make sure your image clearly shows the printed ppm labels ‚Äî the AI reads labels, not peak positions
  • Check that labels aren't cropped or cut off at the edges
  • High-resolution images work best (avoid phone screenshots of screens if possible)
  • Tell the AI the spectrum type if it wasn't detected: This is a 13C NMR in CDCl3
  • Check the confidence column ‚Äî LOW confidence peaks may be misread
How do I start a fresh conversation?
Click "New Chat" in the sidebar. Each chat maintains its own conversation history and file context. Your previous chats are saved and can be accessed from the sidebar.
The export download link isn't working?
Try clicking the link again ‚Äî if the file was still being generated, it may need a moment. If it still doesn't work, try the export command again. Make sure you have an analysis or response to export first (the AI needs content to format into a document).
The AI isn't detecting my spectrum image?
The NMR analysis mode activates when it detects spectrum-related keywords in your message. Make sure to mention the type in your prompt ‚Äî for example, include words like "NMR", "spectrum", "13C", "1H", "ppm", or "DEPT". Simply uploading an image without context may not trigger the specialized spectrum processing.
Response seems slow or loading?
Large files (multi-slide PowerPoints, big PDFs with images) take longer to process because the AI extracts and analyzes all content. NMR analysis also takes slightly longer due to the 5-crop zoom and 3-pass verification process. This is normal ‚Äî the AI is being thorough.
