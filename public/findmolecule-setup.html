Copy to FindMolecule ELN

Set up the Chrome extension and native host to paste structures from GL-ChemDraw into FindMolecule.

1. Download the Chrome extension

This is the GL-ChemDraw Clipboard for FindMolecule extension. It runs in Chrome and forwards structures to the Windows clipboard.

Download extension (.zip)

2. Download the native host

This small program runs on your PC and puts the structure on the Windows clipboard in ChemDraw format so FindMolecule can paste it.

Download native host (.zip)

3. Install both

  1. Native host: Extract the zip, open PowerShell in that folder, run .\install.ps1 -User. (Requires Python 3 with pywin32 and cdx-mol – the script installs them if needed.)
  2. Extension: Extract the zip. In Chrome, go to chrome://extensions, turn on Developer mode, click Load unpacked, and select the extracted folder.
Requirements: Windows, Chrome (or Chromium), Python 3. The native host uses the Windows clipboard.

4. Use it

In GL-ChemDraw: Export → Copy for FindMolecule. Then paste (Ctrl+V) in FindMolecule.
