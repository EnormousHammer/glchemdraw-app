Copy to FindMolecule

For GL-ChemDraw on Vercel (or any URL). Two steps — then paste works like ChemDraw.

Step 1: Install the clipboard helper (one click)

Download and run the installer. No Python or technical setup needed.

Download & run installer

Double-click the downloaded file and follow the prompts. Done.

Step 2: Add the Chrome extension

Download the extension, then load it in Chrome.

Download extension
  1. Extract the zip (right-click → Extract all)
  2. Open Chrome and go to chrome://extensions
  3. Turn on Developer mode (top right)
  4. Click Load unpacked and select the extracted folder

That's it

In GL-ChemDraw: Export → Copy for FindMolecule. Then paste (Ctrl+V) in FindMolecule.

Alternative: Manual install (if you have Python)

If the one-click installer isn't available, use the zip:

Download native host (.zip)

Extract, open PowerShell in that folder, run .\install.ps1 -User

Requirements: Windows, Chrome (or Edge). The clipboard helper runs on your PC.

Extension not working?

1. Use Chrome or Edge. Comet and other Chromium browsers may not find the native host. Try Chrome or Edge first.

2. Run the installer first. The extension needs the clipboard helper. Run GL-ChemDraw-Setup.exe before using the extension.

3. Reload the extension. After installing, go to chrome://extensions and click the refresh icon on the GL-ChemDraw extension.

4. Vercel / URL. The extension runs on *.vercel.app, localhost, and 127.0.0.1. If you use a custom domain, add it to the extension manifest.

5. Check the page. Open GL-ChemDraw, draw a structure, then Export → Copy for FindMolecule. You should see "Copied CDX — paste (Ctrl+V) into FindMolecule" if it worked. If you see "Install the Chrome extension...", the extension or native host isn't connected.
